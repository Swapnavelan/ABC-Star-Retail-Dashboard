# ABC Star Retail Dashboard

Interactive retail analytics dashboard developed using Power BI.

## Features
- Total Sales KPI
- Average Sales Analysis
- Customer Segmentation
- Monthly Sales Trends
- Loyalty Tier Analysis
- Regional Filtering

## Tools Used
- Power BI
- DAX
- Power Query
- CSV Datasets
